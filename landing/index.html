Norsk er eit språk med rik historie og mange dialektar. Landet har to offisielle skriftspråk — bokmål og nynorsk — som begge vert nytta i offentleg samanheng.

I dag er bokmål det mest brukte skriftspråket, medan nynorsk står sterkast på Vestlandet og i indre strøk av landet.

Helt lokalt

Alt kjører på din egen Mac via Ollama. Ingen data sendes til skyen.

Kom i gang

1

Installer Ollama

Last ned ollama.com og installer det på Macen din.

2

Last ned NorMistral

Åpne Terminal og kjør: ollama pull marksverdhei/normistral-it:7b

3

Last ned Ivar Aasen BM

Last ned .dmg-filen nedenfor, åpne den og dra appen til Programmer-mappen.

4

Første gangs åpning (macOS)

macOS blokkerer apper som ikke er signert. Kjør dette i Terminal én gang: xattr -cr "/Applications/Ivar Aasen BM.app" — deretter åpner appen seg normalt.

5

Start og skriv

Åpne appen, skriv tekst og bruk verktøylinjen. Sørg for at Ollama kjører i bakgrunnen.
